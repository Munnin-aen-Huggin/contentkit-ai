## Platform Publishing Checklist

Use this before publishing each piece.

**Social Posts (Instagram, LinkedIn, Twitter/X)**
- [ ] Hook is in the first line (not buried after "Hey everyone!")
- [ ] CTA is specific (not just "follow for more")
- [ ] Hashtags are relevant (3–5 max for LinkedIn, up to 30 for Instagram)
- [ ] Visual is sized correctly for the platform
- [ ] Post is scheduled at optimal time (check your platform analytics)

**Email**
- [ ] Subject line tested (write 3 options, pick the strongest)
- [ ] Preview text is set (first line after subject)
- [ ] All links are working
- [ ] Unsubscribe link is present
- [ ] Sent to a test address before broadcast

**Video (YouTube / Reels / TikTok)**
- [ ] Hook is in the first 3 seconds
- [ ] Caption includes keywords for search
- [ ] Thumbnail (for YouTube) has clear text + face
- [ ] End screen or CTA is included
