<objective>
Build 5 subscriber segments and the win-back automation for cold non-buyer leads.

Purpose: Segments (EMAIL-05) enable targeted broadcasts and audience reporting. The win-back sequence (EMAIL-04) re-engages subscribers who haven't opened emails in 90+ days and haven't purchased, giving them one last chance before cleanup.

Output: 5 Kit segments configured, 1 win-back sequence (3 emails), 1 win-back visual automation active, win-back-candidate tag created.

**Important: 60-day vs 90-day discrepancy.** The Phase 7 requirements specify "60+ days of inactivity" for win-back. Kit's cold subscriber status is fixed at 90 days and cannot be configured. This plan uses Kit's native 90-day cold subscriber definition as the trigger criteria. This is a conservative approximation — subscribers are contacted at 90 days instead of 60, meaning slightly fewer re-engagement opportunities but zero custom infrastructure. The alternative (manual monthly tagging of 60-day inactive subscribers) adds operational burden with marginal benefit. Using 90 days is the recommended approach.
</objective>

<task type="checkpoint:human-action" gate="blocking">
  <name>Task 2: Create 5 segments, win-back sequence, and win-back automation in Kit</name>
  <action>
    Complete the following steps in Kit dashboard.

    **Step A — Create win-back-candidate tag:**
    1. Go to Kit Dashboard -> Grow -> Tags (or Subscribers -> Tags)
    2. Create new tag: `win-back-candidate`
    3. Note: This tag will be manually applied to cold non-buyers via a monthly bulk operation

    **Step B — Create 5 segments:**
    Go to Kit Dashboard -> Subscribers -> Segments -> + New Segment for each:

    1. **Buyers-All**
       - Filter: "Tagged with" = "purchased-starter" OR "Tagged with" = "purchased-full-kit"
       - (Use the "Add a filter" with OR condition between the two tag rules)

    2. **Buyers-Starter**
       - Filter: "Tagged with" = "purchased-starter"

    3. **Buyers-Full-Kit**
       - Filter: "Tagged with" = "purchased-full-kit"

    4. **Leads-Only**
       - Filter: "Not tagged with" = "purchased-starter" AND "Not tagged with" = "purchased-full-kit"

    5. **Cold-Subscribers**
       - Filter: Subscriber status = "Cold" (this is Kit's built-in 90-day inactive status, available in the segment filter dropdown)

    Verify each segment shows a subscriber count (may be 0 for buyer segments if no purchases yet — that's fine).

    **Step C — Create Win-Back Sequence:**
    1. Go to Kit Dashboard -> Send -> Sequences -> + New Sequence
    2. Name: "Win-Back"
    3. Add 3 emails from `downloads/winback-email-copy.md`
    4. Set delays: Email 1 = immediate, Email 2 = 4 days, Email 3 = 4 days
    5. Publish all 3 emails

    **Step D — Create Win-Back Visual Automation:**
    1. Go to Kit Dashboard -> Automate -> Visual Automations -> + New Automation
    2. Start from scratch (or use Kit's "Re-engage cold subscribers" template if available — adapt as needed)
    3. Entry trigger: "Tag is added" -> select "win-back-candidate"
    4. Add action: "Add to email sequence" -> select "Win-Back"
    5. Name: "Win-Back Cold Leads"
    6. Set to Active/Live

    **Step E — Document the monthly win-back process:**
    The win-back-candidate tag must be applied manually on a monthly basis to eligible subscribers. The process is:
    1. Go to Kit -> Subscribers -> use the "Cold-Subscribers" segment
    2. Cross-reference with "Leads-Only" segment (subscribers must be in BOTH — cold AND not a buyer)
    3. Select eligible subscribers -> Bulk Actions -> Add Tag -> "win-back-candidate"
    4. This triggers the Win-Back automation for each tagged subscriber

    Note: This is a manual monthly operation, not automated. Kit cannot natively trigger automations from segment membership.
  </action>
