<task type="checkpoint:human-action" gate="blocking">
  <name>Task 2: Create sequences and visual automations in Kit</name>
  <action>
    Using the email copy from `downloads/onboarding-email-copy.md`, set up both onboarding flows in Kit.

    **Step A — Create Starter Onboarding Sequence:**
    1. Go to Kit Dashboard -> Send -> Sequences -> + New Sequence
    2. Name: "Starter Onboarding"
    3. Add 5 emails from the Starter section of onboarding-email-copy.md
    4. Set delays between emails: Email 1 = immediate, Email 2 = 2 days, Email 3 = 3 days, Email 4 = 4 days, Email 5 = 5 days
    5. Publish the sequence (set all emails to Published/Active)

    **Step B — Create Full Kit Onboarding Sequence:**
    1. Kit Dashboard -> Send -> Sequences -> + New Sequence
    2. Name: "Full Kit Onboarding"
    3. Add 5 emails from the Full Kit section of onboarding-email-copy.md
    4. Set same delay structure: immediate, +2, +3, +4, +5 days
    5. Publish the sequence

    **Step C — Create Starter Onboarding Visual Automation:**
    1. Go to Kit Dashboard -> Automate -> Visual Automations -> + New Automation
    2. Start from scratch (blank)
    3. Entry trigger: "Tag is added" -> select "purchased-starter"
    4. Add action: "Add to email sequence" -> select "Starter Onboarding"
    5. Name the automation: "Starter Onboarding"
    6. Set automation to Active/Live

    **Step D — Create Full Kit Onboarding Visual Automation:**
    1. Kit Dashboard -> Automate -> Visual Automations -> + New Automation
    2. Start from scratch
    3. Entry trigger: "Tag is added" -> select "purchased-full-kit"
    4. Add action: "Add to email sequence" -> select "Full Kit Onboarding"
    5. Name: "Full Kit Onboarding"
    6. Set to Active/Live

    **Verification in Kit:**
