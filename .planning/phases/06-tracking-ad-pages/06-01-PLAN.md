  <name>Task 3: Add sessionStorage deduplication guard to thank-you.html</name>
  <files>thank-you.html</files>
  <action>
    Wrap the conversion tracking IIFE at the bottom of thank-you.html (lines ~141-175, inside the script tag) in a sessionStorage guard. The guard must be the first thing inside the IIFE — before any tier detection or event firing.

    Current IIFE structure:
    ```
    (function() {
      var params = new URLSearchParams(window.location.search);
      var tier = params.get('tier') || 'unknown';
      var value = tier === 'starter' ? 147 : tier === 'full' ? 499 : 0;
      // ... Google Ads event ...
      // ... Meta Pixel event ...
      // ... Plausible event ...
    })();
    ```

    New IIFE structure — insert guard as FIRST block inside the function:
    ```
    (function() {
      // SessionStorage deduplication — only fire conversions once per browser session
      var CONVERSION_KEY = 'ck_conversion_fired';
      if (sessionStorage.getItem(CONVERSION_KEY)) {
        return; // Already fired this session — exit without re-firing
      }
      sessionStorage.setItem(CONVERSION_KEY, '1');

      // Detect tier from URL param (?tier=starter or ?tier=full)
      var params = new URLSearchParams(window.location.search);
      var tier = params.get('tier') || 'unknown';
      var value = tier === 'starter' ? 147 : tier === 'full' ? 499 : 0;

      // Google Ads Conversion (real ID already replaced by Task 2)
      if (typeof gtag === 'function') {
        gtag('event', 'conversion', {
          'send_to': 'AW-REALID/REALLABEL',
          'value': value,
          'currency': 'USD',
          'transaction_id': ''
        });
      }

      // Meta Pixel Purchase Event
      if (typeof fbq === 'function') {
        fbq('track', 'Purchase', {
          value: value,
          currency: 'USD',
          content_name: 'ContentKit AI ' + (tier === 'starter' ? 'Starter Kit' : tier === 'full' ? 'Full Kit' : 'Unknown'),
          content_ids: [tier],
          content_type: 'product'
        });
      }

      // Plausible Purchase Event
      if (window.plausible) {
        plausible('Purchase', { revenue: { currency: 'USD', amount: value }, props: { tier: tier } });
      }
    })();
    ```

    Also update the HTML comment above the script from:
    "<!-- Conversion Tracking — fires on page load -->"
    to:
    "<!-- Conversion Tracking — fires once per session (sessionStorage guard prevents duplicate fires on refresh) -->"

    IMPORTANT: Use sessionStorage (not localStorage). sessionStorage clears when the tab closes — a new purchase in a new tab fires correctly. localStorage would persist across sessions and block future legitimate purchases.

    Preserve all existing event data values (value, currency, content_name, content_ids, content_type, transaction_id). Do not remove any event or change any values — only add the guard block at the top of the IIFE.
  </action>
