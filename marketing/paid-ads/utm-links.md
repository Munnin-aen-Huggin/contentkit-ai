## UTM Parameter Reference

| Parameter | Description |
|---|---|
| `utm_source` | Traffic source: `google`, `facebook` |
| `utm_medium` | Medium: `cpc` (cost per click) |
| `utm_campaign` | Campaign name: `search_ai_marketing`, `cold_conversions`, `retargeting`, `lead_gen` |
| `utm_content` | Ad/ad group identifier for A/B testing |

## Notes

- All UTM parameters are automatically captured by Plausible Analytics
- ConvertKit form hidden fields capture `utm_source`, `utm_medium`, and `utm_campaign` from URL params
- Google Ads auto-tagging (`gclid`) works alongside manual UTM tagging
- Meta auto-tagging (`fbclid`) works alongside manual UTM tagging
